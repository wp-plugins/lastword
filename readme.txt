== Description ==

Allows users to see the latest update from lastword.at and to update the site directly from your sidebar granting you a referral backlink on lastword.at as well as all other wordpress sites that run this plugin.

== Installation ==

1. Upload `lastword.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `lastword.at` in your sidebar in the "widgets" menu
